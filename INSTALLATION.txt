🎯 WHAT CHANGED:
═════════════════════════════════════════════════════════════════════════════

FROM                          TO
────                          ──
Next.js                  →    React + Vite
TypeScript (TSX)         →    JavaScript (JSX)
File-based routing       →    React Router v6
API routes (/app/api)    →    In-memory db.js
next/link                →    React Router links
next/navigation          →    useNavigate hook
Server components        →    Client components
